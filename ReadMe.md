# Js基础培训PPT

> 公司培训使用，因不涉及相关商业机密，可网上共享，以技术为出发点，把作者知道的一些零散知识点重新组装，PPT内含有对部分物件的主观臆断，如果冒犯请多包涵
> 本文以Js知识点概要为基础，重在提升对于Js学习，而非重在练习，基础练习可自行网上寻找教程进行完成。因篇幅有限，文中只包含目前主流Js 60%的观点。
